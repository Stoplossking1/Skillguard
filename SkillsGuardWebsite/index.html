Step 1

Point at any repo

Pass a GitHub URL or local path. SkillsGuard clones to a temp sandbox and begins static analysis immediately.

Step 2

Two layers of analysis

Layer 1 runs fast: red flags, capabilities, GitHub Actions, and inspectability in seconds. Layer 2 goes deep: YARA pattern matching, Python AST taint tracking, and optional LLM semantic analysis on the skill manifest.

Start Scanning

Decide with evidence

Every finding includes severity, confidence, file path, line number, evidence excerpt, and remediation guidance. A meta-engine filters false positives so you only see what matters.
